# AI-Based Traffic Signal Optimization (Dubai Case Study)

## Overview
Urban traffic congestion is a major challenge in modern cities. Traditional fixed-time traffic signals are inefficient under dynamic traffic conditions.

This project proposes an AI-based approach to optimize traffic signal timing using machine learning techniques.

---

## Objectives
- Reduce traffic delay at intersections
- Improve traffic flow efficiency
- Compare traditional vs optimized signal control

---

## Dataset
- Synthetic traffic dataset generated using Python
- Features:
  - Traffic volume
  - Time of day
- Simulated delay based on traffic conditions

---

## Methodology
1. Created synthetic traffic dataset  
2. Built baseline model (fixed signal timing)  
3. Applied Machine Learning (Random Forest)  
4. Designed optimized signal timing strategy  
5. Compared performance before and after optimization  

---

## Results

- **Average Delay (Before):** ~28.7 seconds  
- **Average Delay (After):** ~16–18 seconds  
- **Improvement:** **41.5% reduction in delay**

---

## Visualization
Histogram comparison of delay before and after optimization shows a clear shift toward lower delays.
